### **How Modern Software Products Are Built Using Jira & SDLC Methodologies**

#### **1. Software Development Life Cycle (SDLC)**

SDLC is a structured process for developing software with different phases:

1. **Requirement Analysis** - Gather and document requirements.
2. **Planning** - Define scope, timeline, and resources.
3. **Design** - Create architecture and design documents.
4. **Development** - Write code and implement features.
5. **Testing** - Conduct unit, integration, and system testing.
6. **Deployment** - Release software to production.
7. **Maintenance** - Provide support, fix bugs, and update software.

#### **3. Scrum Framework (Agile Methodology)**

Jira is heavily used in **Scrum**, which follows:

1. **Sprint Planning** - Define tasks for 1-4 week sprints.
2. **Daily Standups** - Short meetings for progress updates.
3. **Sprint Execution** - Developers complete tasks.
4. **Sprint Review** - Demo features to stakeholders.
5. **Sprint Retrospective** - Analyze what went well and what can improve.
